# Backend Roadmap

## V0.1 API Skeleton

- Create FastAPI project structure.
- Add health, compute, and render endpoints.
- Define Pydantic schemas.
- Add placeholder services and engine modules.
- Document architecture and API contracts.

## V0.2 Symbolic Math Computation

- Implement expression parsing with SymPy.
- Add derivative and integral computation.
- Add gradient, divergence, and curl computation.
- Return plain-text and LaTeX results.
- Add focused pytest coverage.

## V0.3 Sampling And Visualization Data

- Add NumPy sampling for scalar fields and vector fields.
- Return frontend-friendly numeric data.
- Add output JSON data files.
- Define sampling validation rules.

## V0.4 Manim Rendering

- Add render task preparation.
- Implement Manim runner integration.
- Create initial scene classes.
- Generate video and image outputs.
- Add render status tracking.

## V1.0 Frontend Integration-Ready Backend

- Stabilize API contracts.
- Support frontend compute and render workflows.
- Improve local setup and packaging.
- Add robust errors and logs.
- Prepare for a complete Calculus Visualizer handoff.
