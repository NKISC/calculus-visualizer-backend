# Sprint Plan

## Sprint 1: FastAPI Skeleton And Schemas

Goals:

- Create backend project structure.
- Define FastAPI app entry point.
- Add health, compute, and render endpoints.
- Define Pydantic request and response schemas.
- Add example request payloads.
- Document architecture and API contracts.

Status: current skeleton.

## Sprint 2: Math Engine Implementation

Goals:

- Implement safe expression parsing.
- Add derivative and integral support.
- Add gradient, divergence, and curl support.
- Add unit tests for symbolic computation.
- Define clear error handling for invalid expressions.

## Sprint 3: Visualization Data Generation

Goals:

- Add NumPy sampling for scalar fields.
- Add NumPy sampling for vector fields.
- Shape numeric data for frontend plotting.
- Add tests for sampling boundaries and density.
- Save optional JSON data outputs.

## Sprint 4: Manim Rendering Integration

Goals:

- Implement render manager and Manim runner.
- Build initial Manim scenes for each task type.
- Add render quality and format handling.
- Save generated video and image outputs.
- Add task status tracking for render jobs.

## Sprint 5: Frontend Integration And Packaging

Goals:

- Stabilize API contract with frontend teammate.
- Confirm output path conventions.
- Add local development scripts.
- Improve documentation for handoff.
- Prepare a backend package that is ready for local app integration.
